# Inhibitory-Control-SCZ-MDD-Data

Data for paper "Shared and disorder-specific computational mechanisms of inhibitory control in schizophrenia and major depressive disorder"

## Overview

Behavioral data from Stroop and Go/NoGo tasks across three groups:
- HC: Healthy Controls (n=86)
- MDD: Major Depressive Disorder (n=86)
- SCZ: Schizophrenia (n=87)
